# 🍕 Pizza Sales Analysis Using SQL Server

## 📊 Project Overview
Comprehensive analysis of pizza sales data using SQL Server to identify revenue trends, customer behavior, and business performance metrics.

## 🎯 Key Metrics
- **Total Revenue**: $817,860
- **Total Orders**: 21,350
- **Pizzas Sold**: 49,574
- **Average Order Value**: $38.31
- **Avg Pizzas/Order**: 2.32
- **Peak Day**: Friday (3,538 orders)

## 🛠️ Tools
- SQL Server
- SQL Server Management Studio (SSMS)

## 📋 Analysis Sections
- **A**: KPI Calculations (Revenue, Avg Order Value, Quantities)
- **B**: Daily Trend Analysis
- **C**: Monthly Trend Analysis
- **D**: Sales % by Pizza Category
- **E**: Sales % by Pizza Size
- **F-L**: Product Rankings (Top/Bottom Performers)

## 💡 SQL Skills
✅ Aggregations (SUM, COUNT, AVG)
✅ GROUP BY & ORDER BY
✅ Date Functions (DATEPART, DATENAME)
✅ DISTINCT & CAST
✅ Complex Filtering
